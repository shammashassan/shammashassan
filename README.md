## Featured Projects

### 💰 [Dime](https://github.com/shammashassan) — Personal Finance Tracker
> Next.js · TypeScript · MongoDB · Better Auth · Tailwind CSS v4 · Resend

Production-ready finance tracker with fully isolated per-user data. The interesting part: 6 authentication methods (email/password, Google OAuth, Magic Link, Passkey/WebAuthn, 2FA with TOTP) and an invite-only onboarding flow with an admin approval queue.

---

### 🧠 [UI Dev Second Brain](https://ui-dev-second-brain.vercel.app) — Developer Resource Platform
> Next.js · TypeScript · MongoDB · Better Auth · shadcn/ui · Vercel

A full-stack platform for design engineers to discover and organise UI resources — components, animations, CSS tools, icon libraries, AI tooling. Built an authenticated management system so users can create categories, tag entries, and keep their stack curated.

---

### 🏢 ERP System — Business Management Platform
> Next.js · TypeScript · Node.js · MongoDB · shadcn/ui · Vercel

End-to-end ERP covering billing, inventory, expenses, employee records, and journal entries with role-based access control. Designed the MongoDB schema, built all the REST APIs, and shipped a responsive dashboard with exportable reports.

---

### 🤖 Career Guidance App — AI-Powered Android App
> Python · Android · NLP · Scikit-learn · REST API

Parses uploaded resumes using NLP, extracts skills and traits, then recommends courses and job roles via a Python classification model served over a REST API. Improved relevance over a keyword-matching baseline through iterative fine-tuning.
